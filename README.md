# DesignProfissional_A1
Projeto criado por gustavruas como parte de avaliação final na matéria de Design.

## Objetivo
Agrupar toda/maioria das atividades feitas esse semestre, provando competência e compromisso para obtenção de conhecimento e também softskills.

### Objetivo da Calculadora em C
1. Demostrar competência no uso de lógica de programação - condicionais, loops, funções, definição de variavéis
2. Softskills - estudar e fazer pesquisa de como cada função matemática funcionam.
### Objetivo do Sistema Gerencial de Alunos
Demostrar competência no uso de:
1. Classes
2. Objetos
3. Funções
### Objetivo do Estudo de algoritmos de Ordenação(Sort)
Demostrar competência no: 
Entendimento e implementação de algoritmos de ordenação.
### Objetivo do Projeto001
Demostrar competência no entendimento básico de: 
1. HTML
2. Css
3. JavaScript.
### Objetivo do Projeto002
Demostrar competência no uso de Materialize.
### Objetivo da provaA1Grupo07
Demostrar competência no uso de Bootstrap.
### Objetivo da Cidade inteligente
Demostrar competência e entendimento de modelos conceituais, lógicos e suas representações, além de código em SQL.
